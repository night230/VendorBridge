# VendorBridge
Procurement &amp; Vendor Management ERP for managing vendors, RFQs, quotations, approvals, purchase orders, invoices, and procurement analytics.

# 🚀 VendorBridge Authentication System

A full-stack authentication system with **Signup, Login, OTP verification, Password Reset, and Social Login (Google + GitHub)** built using:

- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: SQLite
- Email Service: Flask-Mail (Gmail SMTP)

---

## 📌 Features

### 🔐 Authentication
- User Signup with email & password
- Login system with JWT token
- Duplicate user prevention
- Session storage using localStorage

### 📩 OTP System
- OTP sent to email on signup
- OTP verification page (`verify-otp.html`)
- Password reset using OTP

### 🌐 Social Login
- Google Login (Google Identity Services)
- GitHub OAuth login

### 🛡 Security
- JWT-based authentication
- Password reset via OTP
- Backend validation for all requests
